# Changelog

## 1.0.0 (2026-03-13)

- Initial release.
- Tag-based quiz embedding: `{sborniktestov id=NNN}`.
- Optional auto-embed of links to sborniktestov.ru quiz pages.
- Configurable display defaults (title, description, statistics, comments, related quizzes).
- Customisable colours, fonts and layout via tag parameters.
- Automatic iframe height adjustment via postMessage.
- Privacy API implementation declaring external data transmission.
- English and Russian language packs included.
